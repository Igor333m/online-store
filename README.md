# Online shop

## Frontend & Backend

After git clone, run **_npm install_** in both folders.
On backend, add variable.env with prisma endpoint, then run **_npm run deploy_**.

To run *frontend* part, run **_npm run dev_**.


## The Tech Stack of Online Store

* React
  * Next.js
  * Styled Components
  * NProgress
* GraphQL
  * Apollo
  * Prisma server
* Nodejs
* Stripe payments
* Heroku deployment
